Computer Engineer & Creative Developer

I work in artificial intelligence, machine learning, and computer vision, focusing on building practical models and end-to-end systems. I have hands-on experience with deep learning, YOLOv8, CNN-based image classifiers, NLP pipelines, semantic search (MiniLM + FAISS), and API development using FastAPI. My background includes dataset preparation, model training, backend integration, and deploying AI-driven applications that operate reliably with real data.

I continuously improve in both areas and enjoy working on projects that require analytical thinking, hands-on development, and a fast learning approach.
